# Smansys - School Management Admin Dashboard

A modern School Management Admin Dashboard built using the MERN stack. The application provides separate dashboards for Super Admin and School Admin with features for managing schools, students, teachers, and fee-related operations.

## Features

- Secure JWT Authentication
- Super Admin Dashboard
- School Admin Dashboard
- Student Management
- Teacher Management
- School Onboarding
- CSV Import
- Fee Structure Management
- Bulk Email Notifications
- Responsive Dashboard UI

## Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- React Router
- React Hook Form
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Nodemailer
- Multer

## Project Structure

```
smansys/
├── frontend/

```

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
